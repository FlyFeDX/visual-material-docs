### 交互配置

支持点击图形下钻及事件派发

![交互配置](./images/i-11.png)

-   支持点击图形展示弹框或抽屉
-   支持点击图形时,向外派发数值

    ```js
       示例配置:
       X轴字段[x] : paramX
       Y轴字段[y] : paramY
       Y轴字段[s] : paramS

       示例数据:
      [
        {
            "x": "1991",
            "y": 13,
            "s": "s1",
            "t": 1
        },
        {
            "x": "1992",
            "y": 17.8,
            "s": "s4",
            "t": 2
        }
      ]

      点击柱图,对外派发的参数为:
      {paramX:"1991",paramY:13,paramS:"s1"}

      点击线图,对外派发的参数为:
      {paramX:"1992",paramY:17.8,paramS:"s4"}

    ```
