

# 基础面积图(base-area) 组件

## 简述

基础面积图(单系列面积)与折线图类似，均可以展示连续性数据，面积图可以更好展示出数据的趋势、累积，减少等变化 ，组件支持图形的 X/Y 轴，提示信息、数据点、面积等样式设置。
